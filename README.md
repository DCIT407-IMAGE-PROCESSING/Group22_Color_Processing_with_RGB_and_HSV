# Group22 — Color Processing with RGB and HSV

Project scaffold for image color processing experiments and final notebook.

Structure
- `notebooks/` — final deliverable and individual explorations
- `src/` — helper modules only (color conversions, visualization, processing)
- `images/input/` — test images
- `images/output/` — results
- `docs/` — theory and references

Quick start
1. Create a virtual environment: `python -m venv .venv`
2. Activate it, then install: `pip install -r requirements.txt`
3. Open `notebooks/main_project.ipynb` in Jupyter.
